

# Вид формул для фильтра:

- арифметические операции (`+`, `-`, `*`, `/`, `^` для XOR). 
- сравнения (`>`, `<`, `==`, `!=`, `>=`, `<=`).
- логические `and`, `or`.
- унарный `not`.
- значение константы (число или строка (`'2025-01-01'`, `101`, `Российский рубль`)) (даты сравниваются корректно, в формате `'yyyy-mm-dd'`).
